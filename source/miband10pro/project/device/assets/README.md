# Smart Band 10 Pro assets

这里存放 TIME FLIES 的 10 Pro 图片资源。

## 暂定画布

```text
400 × 480
```

## 迁移要求

- 原始资源是 TGA 数据，但文件扩展名为 `.png`。
- 时间数字、日期数字和图标必须保持宽高比。
- 原 192×490 背景不能直接拉伸到 400×480。
- 背景优先采用居中扩展、裁切或重新设计。
- 首版布局使用左侧主时间、右侧数据栏。
